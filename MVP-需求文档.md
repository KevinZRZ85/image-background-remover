# Image Background Remover - MVP 需求文档

> 版本: v1.0 | 日期: 2026-03-20 | 状态: 初稿

---

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | Image Background Remover |
| **产品类型** | AI 图像处理工具 / SaaS |
| **核心价值** | 快速、免费、一键去除图片背景 |
| **目标市场** | 全球开发者、电商卖家、设计师 |

### 1.1 产品定位
面向出海开发者的轻量级 AI 抠图工具，通过 Cloudflare Workers 部署，降低基础设施成本，快速验证市场需求。

## 2. 目标用户

| 用户画像 | 场景 | 痛点 |
|----------|------|------|

## 3. MVP 功能清单

### 3.1 核心功能 (P0 - 必须)

| 功能 | 描述 | 验收标准 |
|------|------|----------|
| **图片上传** | 支持拖拽或点击选择本地图片 | 支持 JPG/PNG/WebP，大小 ≤10MB |
| **URL 输入** | 支持输入图片地址 | 自动识别并加载 |
| **AI 抠图** | 调用 Remove.bg API 处理 | 5秒内返回结果 |
| **结果预览** | 显示处理后的图片 | 清晰展示透明背景 |
| **下载保存** | 一键下载 PNG 结果 | 文件名默认为 `original-rmbg.png` |

### 3.2 增强功能 (P1 - 后续版本)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **批量处理** | 一次上传多张图片 | P1 |
| **历史记录** | 本地存储处理历史 | P1 |
| **API 开放** | 供其他开发者调用 | P2 |
| **对比模式** | 前后对比滑块 | P2 |

## 4. 用户流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  访问网站   │ ──▶ │  上传图片   │ ──▶ │  AI 处理中  │ ──▶ │  下载结果  │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
      │                   │                   │                   │
      ▼                   ▼                   ▼                   ▼
   Landing Page      Drag & Drop        Loading Spinner      PNG File
                    or URL Input       (5s max)
```

### 4.1 关键交互

1. **首页**：简洁的单一页面，上传区域明显
2. **上传**：拖拽区域 + 点击选择 + URL 输入（三选一）
3. **处理**：显示加载动画，不超过 10 秒
4. **结果**：显示透明背景图片，提供下载按钮
5. **错误**：友好提示错误信息，如 API 配额用尽

## 5. 技术架构

### 5.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端** | Vanilla HTML/JS | 轻量、无需构建 |
| **后端** | Cloudflare Workers | Serverless，按调用付费 |
| **AI 服务** | Remove.bg API | 按调用付费 ($0.009/张) |
| **域名** | Cloudflare DNS | 免费 CDN |

### 5.2 部署架构

```
用户浏览器
      │
      ▼
Cloudflare CDN (全球加速)
      │
      ▼
Cloudflare Workers (处理请求)
      │
      ├── 检查参数
      │
      ▼
Remove.bg API (AI 抠图)
      │
      ▼
返回结果给用户
```

### 5.3 成本估算

| 项目 | 免费额度 | 超出费用 |
|------|----------|----------|
| Cloudflare Workers | 100,000 次/天 | $0.13/百万次 |
| Remove.bg API | 50 次/月 | $0.009/张 |
| 域名/SSL | 免费 | - |

**预估**：MVP 阶段 0 成本验证，500 用户以内无需付费。

## 6. API 设计

### 6.1 内部 API

```
POST /api/remove
Content-Type: multipart/form-data

参数:
  - image: File (可选)
  - imageUrl: String (可选)

响应:
{
  "success": true,
  "resultUrl": "data:image/png;base64,...",
  "creditsRemaining": 45
}
```

### 6.2 错误码

| 错误码 | 描述 |
|--------|------|
| 400 | 无效请求 / 未提供图片 |
| 403 | API Key 未配置 |
| 429 | API 配额用尽 |
| 500 | 服务器错误 |

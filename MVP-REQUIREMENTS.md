# Image Background Remover - MVP 需求文档

## 1. 项目概述

**产品名称**: Image Background Remover (图片背景移除工具)

**产品定位**: 免费在线 AI 图片背景移除工具，为用户提供快速、准确的背景移除服务

**目标用户**:
- 电商卖家（需要产品白底图）
- 设计师（快速抠图需求）
- 社交媒体用户（头像/照片美化）
- 普通用户（证件照、简历照片等）

**核心价值**:
- 无需专业技能，一键自动处理
- 快速高效，秒级出图
- 免费使用，无需注册

---

## 2. 功能需求

### 2.1 核心功能（MVP 必须）

#### F1: 图片上传
- **描述**: 用户可通过拖拽或点击上传图片
- **输入**: PNG/JPG/WEBP 格式图片，最大 10MB
- **输出**: 图片预览
- **验证**: 
  - 文件格式检查
  - 文件大小限制
  - 错误提示

#### F2: 背景移除
- **描述**: 调用 AI API 自动移除图片背景
- **输入**: 用户上传的图片
- **处理**: 通过 Remove.bg API 处理
- **输出**: 透明背景 PNG 图片
- **状态**: 显示处理进度（加载动画）

#### F3: 结果预览
- **描述**: 并排展示原图和处理后的图片
- **功能**:
  - 左侧：原始图片
  - 右侧：去背景图片（棋盘格背景显示透明度）
  - 响应式布局（移动端上下排列）

#### F4: 图片下载
- **描述**: 用户可下载处理后的图片
- **格式**: PNG（保留透明通道）
- **命名**: `原文件名-no-bg.png`
- **触发**: 点击"Download PNG"按钮

#### F5: 重新上传
- **描述**: 处理完成后可上传新图片
- **功能**: 清空当前状态，返回上传界面

### 2.2 非功能需求

#### NF1: 性能
- 图片处理时间 < 10 秒（取决于 API）
- 页面加载时间 < 2 秒
- 支持 Edge Runtime（全球加速）

#### NF2: 兼容性
- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 移动端响应式设计
- 最小屏幕宽度 320px

#### NF3: 安全与隐私
- 图片仅在内存中处理，不存储
- HTTPS 加密传输
- 不收集用户数据

#### NF4: SEO 优化
- 语义化 HTML
- Meta 标签优化
- 结构化数据（FAQ）
- 关键词优化

## 3. 技术架构

### 3.1 技术栈
- **前端框架**: Next.js 14 (App Router)
- **运行时**: Edge Runtime
- **部署平台**: Cloudflare Pages
- **AI 服务**: Remove.bg API
- **样式**: 原生 CSS（无依赖）

### 3.2 系统架构
```
用户浏览器
    ↓
Next.js 前端 (Cloudflare Edge)
    ↓
API Route (/api/remove-bg)
    ↓
Remove.bg API
    ↓
返回处理后的图片（内存）
```

### 3.3 数据流
1. 用户上传图片 → FormData
2. 前端发送 POST 请求 → `/api/remove-bg`
3. 后端转发到 Remove.bg API
4. 接收 PNG 结果 → 直接返回（不存储）
5. 前端展示结果 → Blob URL
6. 用户下载 → 触发浏览器下载

## 4. 用户流程

### 4.1 主流程
```
访问网站
    ↓
上传图片（拖拽/点击）
    ↓
等待处理（3-8秒）
    ↓
查看对比结果
    ↓
下载 PNG / 上传新图片
```

### 4.2 异常流程
- **文件格式错误**: 提示"请上传图片文件"
- **文件过大**: 提示"图片必须小于 10MB"
- **API 失败**: 提示"处理失败，请重试"
- **API 额度用尽**: 提示"服务暂时不可用"

## 5. UI/UX 设计

### 5.1 设计原则
- **简洁**: 单页应用，无多余步骤
- **直观**: 拖拽上传，一目了然
- **反馈**: 清晰的状态提示（上传中、处理中、完成）
- **对比**: 原图与结果并排展示

### 5.2 色彩方案
- 主色调: 深色模式（黑灰）
- 强调色: 紫蓝色（#6366f1）
- 成功色: 绿色（下载按钮）
- 背景: 渐变标题，棋盘格透明区

### 5.3 关键页面元素
1. **标题区**: 产品名 + 简短描述
2. **上传区**: 虚线边框，拖拽提示
3. **预览区**: 左右分栏（原图 vs 结果）
4. **操作区**: 下载 + 重新上传按钮
5. **特性区**: 3 个卖点卡片（快速/精准/安全）
6. **FAQ 区**: 常见问题（SEO）

## 6. API 集成

### 6.1 Remove.bg API
- **文档**: https://www.remove.bg/api
- **免费额度**: 50 次/月
- **付费方案**: 
  - $0.20/张（按需）
  - $9/月（40 张）
  - $99/月（1000 张）

### 6.2 API 调用
```
POST https://api.remove.bg/v1.0/removebg
Headers:
  X-Api-Key: <your_key>
Body:
  image_file: <file>
  size: auto
```

### 6.3 错误处理
- 402: API 额度用尽
- 429: 请求过于频繁
- 500: 服务器错误
